# 部署

本项目基于Python开发请在运行之前完成对依赖安装.

如果你是Linux用户可以使用以下命令进行环境的配置
    
    ```bash
    make
    ```

该命令将会创建一个虚拟环境，并在其中并在其中依赖的安装，请在安装完成后手动的激活虚拟环境。

如果你是Windows用户的话, 你也可以使用以下命令手动安装所需的包：

    ```bash
    pip install -r requirements.txt
    ```

# 使用方法

你可以使用以下命令运行项目演示：

    ```bash
    make demo
    ```
